JAVASCRIPT & THE DOM

HTML VS CSS

Almost always mentioned in Web Development, HTML and CSS are mandatory tools used to create user intuitive and fuctioning websites but, there is a difference. HTML is like the skeletal structure of a building. It provides the necessary framework, like beams, foundations or rooftops. CSS on the other hand is the design aspects of the building. It accounts for paint or interior design. Without HTML or CSS a website would be difficult to creat.

CONTROL FLOW AND LOOPS

Control flow is a natural order in which a program executes or evaluates a set of statements and instructions. This order can be navigated using conditional statements or loops but will still abide by the Control Flow.

An easy way to understand this would be cleaning the house;

if (house == dirty) {
houseClean();
} else {
moreMess();
}

The above example uses a condition statement to check if something is true. If “house” is dirty then execute the “houseClean()” method. In any other case call the “moreMess()” method.

Loops are good way to keep a program running until a condition is met. A good example would gas in a lawnmower.

do {
mowLawns();
} while (gasTank == full)

Here we see that the “mowLawns” method will be invoked until “gasTank” !== full.

WHAT IS THE DOM?

The DOM is a tree of objects that stem from a parent object called the window. As an acronym it stands for Document Object Model.

The DOM is basically used to interact with different elements on a webpage. It helps with styling, adding or removing elements and provides more malleable JavaScript interactivity.

ARRAYS VS OBJECTS

An Array[] is an arrangement of quantities or elements in rows and columns. Essentially an Array is a matrix. An Object is a data construct that provides a description of properties. Although Objects and Arrays are similar, they are different.

Try to think of an Array as a shopping list.

var list = [milk, eggs, butter, bread, carrots, tomatoes];

Here we see 6 elements in the array. Unlike people, Arrays count from index 0. This means the milk would equal index [0], eggs would equal [1] and so on and so forth until tomatoes equals [5].

An Object on the other hand is a list of properties which are referenced by keys instead of and index. An example would be;

var animal = {
“dog”
“cat”
“horse”
“cow”
“chicken”
}

In order to access an object property, we reference them using a “.” like so;

"animal.dog" or "animal.chicken"

Objects can be used in very creative programming techniques and some objects can depend on other objects. This is known as object orientated programming.

FUNCTIONS AND WHY THEY ARE USEFUL

A function is a set of rules or commands that dictate how the control flow works. They are used to implement creative tasks. A function is also known as a method.

Functions take a parameter list which is input. We then use statements in our function to dictate how the parameter or input is used. The function then returns a value based on our statement.

We can also call functions within functions. This allows for less duplicate code, best practice and easy readability.
